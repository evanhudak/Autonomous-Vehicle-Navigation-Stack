# Autonomous-Vehicle-Navigation-Stack
Python-based autonomous vehicle navigation and control software integrating camera, LiDAR, and actuator subsystems for real-time testing and validation.

This repository contains a Python-based navigation and control stack developed for a Boeing-sponsored autonomous vehicle capstone project at Northern Arizona University. The software integrates camera, LiDAR, sensor-fusion, and actuator subsystems to support real-time path following, command generation, testing, and validation. The project emphasizes embedded-style software development, subsystem integration, debugging, and safety-focused control logic.
