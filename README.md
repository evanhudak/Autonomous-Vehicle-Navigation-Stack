# Autonomous-Vehicle-Navigation-Stack
Python-based autonomous vehicle navigation and control software integrating camera, LiDAR, and actuator subsystems for real-time testing and validation.

This repository contains a Python-based navigation and control stack developed for an autonomous vehicle capstone project sponsored by Boeing at Northern Arizona University. The system integrates camera, LiDAR, sensor-fusion, and actuator subsystems to generate real-time steering, throttle, and braking commands for testing and validation.

The project focuses on hardware-software integration, real-time control, subsystem communication, and safety-oriented validation in a live autonomous vehicle environment.
